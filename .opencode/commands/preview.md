Manage preview server: start, stop, status check.

User request: $ARGUMENTS

## Sub-commands interpretation

- `start` - Start development server
- `stop` - Stop server
- `restart` - Restart server
- `check` - Health check
- (empty) - Show current status

## Actions

Based on the user's request, perform the appropriate action:
- Start the dev server and show URL
- Stop running server
- Check server health
- Show current status
